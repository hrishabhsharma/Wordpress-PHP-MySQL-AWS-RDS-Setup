Requirements

Installing apache webserver
dnf install httpd -y

1) epel-release
dnf install https://dl.fedoraproject.org/pub/epel/epel-release-latest-8.noarch.rpm -y


2) Remi repo is a free and stable YUM repository mainly for the PHP 
dnf install https://rpms.remirepo.net/enterprise/remi-release-8.rpm -y


3) Installing php-7.4
dnf module install php:remi-7.4

4) dnf install php-mysqlnd -y

5) To download the WordPress installation package
wget https://wordpress.org/latest.tar.gz

6) extracting package and copying wordpress folder to document root /var/www/html
tar -xvzf latest.tar.gz -C /var/www/html 

7)Permissions:
chown -R apache:apache /var/www/html/wordpress
chcon -t httpd_sys_rw_content_t /var/www/html/wordpress -R

8) Creating MySQL Database on AWS using RDS Service

9)MYSQL Installation
dnf install mysql -y

10)MySQL login
mysql -h endpoint -u -p
